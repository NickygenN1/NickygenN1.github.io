What is Machine Learning ?


Machine learning is a result from computer and algorithm (math), can find a pattern of problem
(in math is called function) without human to program it.

It learn by many of data then find the pattern :

- this picture is a cat or dog
- how bitcoin price next day (if i can do, i won't tell you)
- can cluster ralative data with other and seperate what is abnormal
- how many hour for study is get a best grade



Type of Machine Learning


1. Supervised learning : have x predict y
- regression : predict the number(price of BTC in next monday, what grade if i study 1 hour per day)
- classification : categorical (this is picture of cat or dog)


2. Unsupervised learning : have only x (find relative of x)
- clustering : i want group of 3 of customer that have somthing relative.
- non-clustering : what is abnormal from the other


for more ML has Reinforcement learning : reward and punish, computer learn to get the most reward (alphago learn to play go for million time a day)
